How this page is made

This page is made using a web component called html-import it takes a src attribute as an html snippet to include in the page and renders it instead of the import statement. The result is a semantic html page that takes advantage of modularization and progressive loading.

The component is quite lightweight: 

class ImportHTML extends HTMLElement {
  constructor() {
    self = super();
    const src = this.getAttribute("src");
    fetch(src)
      .then(data => data.text())
      .then(html => {
        this.outerHTML = html;
      });
  }
}
customElements.define("html-import", ImportHTML);

The intention is to create a lightweight modular html experience similar to with a HTML preprocessor or a static site generator. It is a client side site generator of sorts. Inspiration comes from the HTML imports spec The component will cause network requests for each include so use sparingly.

Example:

<html lang="en"> <body> <script src="/script.js" defer></script> <html-import src="header.html"></html-import> <html-import src="nav.html"></html-import> <main> <html-import src="about.html"></html-import> <html-import src="principles.html"></html-import> <html-import src="how-this-page-is-made.html"></html-import> </main> </body> </html>
